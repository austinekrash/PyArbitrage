# PyArbitrage

PyArbitrage is a python script to obtain the percentage of discount between all pairs of cryptocurrency and arbitrage between the major crypovalue exchanges.

## Supported exchanges:

- Binance
- Bittrex
- Bitfinex
- Cex
- Poloniex
